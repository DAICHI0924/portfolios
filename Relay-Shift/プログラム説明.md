# プログラム説明

## 開発環境・使用技術
### 開発環境
- **エンジン**: Unity 6000.3.11f1
- **言語**: C#
- **IDE**: Visual Studio Code

### 使用ライブラリ
- **LitMotion**: アニメーション処理
- **UniTask**: 非同期処理
- **Cinemachine**: カメラ挙動・切り替え制御
- **TNRD.SerializableInterface**: インターフェースのシリアライズ

---

## 担当範囲
- ゲームシステム設計・実装
- プレイヤー（ランナー）切り替え・バトンパス制御の実装
- スローモーション（タイムスケール）制御システムの実装
- Cinemachineによるカメラ追従・切り替えロジック
- バトン落下アニメーションの実装

---

## コアシステム

### 1. タイムスケール・スローモーション制御
バトンパス時に全体の時間をスローにして演出を強化し、パス終了後に通常速度に戻します。
- `TimeScaleManager` で `ISlowable` 実装オブジェクトの時間を管理
- `PlayerTimeScale` でプレイヤー移動とアニメーションを同期制御

### 2. バトンパス・ランナー制御
バトン受け渡しの衝突検知と次のランナーへの切り替えを扱います。
- `BatonPassPublisher` で接触を検出しイベント発火
- `RunnerChanger` でバトンを次のランナーに装着

### 3. カメラ制御
Cinemachine の優先度切り替えで、演出に応じた視点を動的に変更します。
- `CameraManager` でスタート、パス成功、ゲームオーバー時のカメラ遷移を管理

### 4. バトン演出
ランナー脱落時のバトン飛翔を物理的に再現します。
- `BatonAnimator` で親子解除、重力有効化、飛翔と回転を制御
